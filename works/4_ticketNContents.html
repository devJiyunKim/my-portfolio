티켓앤 APP 서비스 컨텐츠 제작

HTML / CSS / jQuery / Photoshop / Illustrator

기여도 100% (디자인 / 퍼블리싱)

스마트 티켓 어플 ticketN 내에 있는 페이지와 컨텐츠들을 기획 및 제작했습니다.
티켓을 등록한 고객에 한하여 공연 당일 제공되는 서비스 컨텐츠로써, 각 컨텐츠는 관리자 페이지를 통해 등록 및 수정 가능한 페이지입니다.
각 페이지는 웹·앱 개발자와 협업하여 제작하였으며 매 공연마다 해당 아트웍을 활용한 셋리스트, 아티스트 포토, 공연장 맵 이미지를 제작했습니다.

ticketN 어플리케이션 내 티켓 하단에 있는 버튼을 클릭하면 나오는 컨텐츠입니다.

셋리스트

해당 공연 회차에 제공되는 셋리스트입니다.
영수증 형태의 이미지로 각 회차에 따른 공연 셋리스트가 보여집니다.

스테이지 포토

해당 공연 회차에 제공되는 스테이지 포토입니다.
포토카드 형태로 공연 로고 및 회차, 관련 사진과 함께 유저의 이름과 아티스트의 메세지가 보여집니다.

공연장 맵

해당 공연의 주변 지도를 볼 수 있는 공연장 맵입니다.
일러스트로 작업한 맵 이미지를 활용, html / css / javascript로 이미지 매핑을 통해 팝업창이 보여지도록 만들었습니다.
